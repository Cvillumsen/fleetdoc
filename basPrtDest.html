Function: adhRetrieveDevNames(strName As String, intType As Integer, varDM As Variant) As Variant

The adhRetrieveDevNames function retrieves the prtDevNames property from a specified form or report, allowing the modification of the printer settings.

Description:

This function opens the specified form or report and retrieves the prtDevNames property, which contains the printer device names. If the printer settings have not been defined, it retrieves the default device's DevMode structure.

Parameters:

  • strName: The name of the form or report to retrieve the printer settings from.
  • intType: Specifies whether the object is a form (acForm) or a report (acReport).
  • varDM: A Variant that will store the DevMode structure of the printer.

Returns:

Returns a Variant containing the prtDevNames string, or Null if it could not retrieve the string.

Example:

To retrieve the printer settings for a form named "frmReport": 

Dim varDM As Variant
Dim devNames As Variant

devNames = adhRetrieveDevNames("frmReport", acForm, varDM)

If Not IsNull(devNames) Then
    Debug.Print "Printer Device Names: " & devNames
End If

Private Function: GetObjectType(obj As Object) As Integer

The GetObjectType function determines whether the provided object is a form or a report.

Description:

This function attempts to identify if the given object is a form or a report. It assumes that only forms and reports are passed to it, and does not handle other object types. The function returns the appropriate type constant (acForm or acReport).

Parameters:

  • obj: A reference to the object (either a form or report) whose type needs to be determined.

Returns:

Returns acForm if the object is a form, and acReport if it is a report.

Example:

To determine the type of an object: 

Dim objType As Integer
objType = GetObjectType(Forms!frmExample)

If objType = acForm Then
    Debug.Print "This is a form."
ElseIf objType = acReport Then
    Debug.Print "This is a report."
End If
