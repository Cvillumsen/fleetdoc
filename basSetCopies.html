Functions in basSetCopies.bas

Function: SetCopies(obj As Object, intCopies As Integer)

The SetCopies function sets the number of copies for a form or report in design mode by modifying the prtDevMode property.

Description:

This function allows you to set the number of copies a report or form will print by modifying the printer's DEVMODE structure. It can only be used when the report or form is open in design mode. If the report or form has no existing printer settings, the function uses the default printer's settings.

Parameters:

  • obj: A reference to the form or report where the number of copies will be set.
  • intCopies: An integer representing the number of copies to print.

Returns:

This function does not return a value directly, but it modifies the prtDevMode property of the form or report.

Example:

To set the number of copies to 3 for a report named "rptCompany": 

' Example usage
SetCopies Reports!rptCompany, 3

This will set the number of copies to 3 for the "rptCompany" report.

Function: SetCopies2(obj As Object, intCopies As Integer)

The SetCopies2 function is an alternative method for setting the number of copies for a form or report by using the adhRetrieveDevMode and adhSetDevMode helper functions.

Description:

This function simplifies the process of setting the number of copies by first retrieving the current DEVMODE structure with adhRetrieveDevMode, then modifying the number of copies, and finally applying the changes using adhSetDevMode. It is a more streamlined version compared to the SetCopies function and performs error checking.

Parameters:

  • obj: A reference to the form or report where the number of copies will be set.
  • intCopies: An integer representing the number of copies to print.

Returns:

Returns True if the number of copies was successfully set, otherwise returns False.

Example:

To set the number of copies to 3 for a report named "rptCompany": 

Dim result As Boolean
result = SetCopies2(Reports!rptCompany, 3)

If result Then
    Debug.Print "Number of copies set successfully."
Else
    Debug.Print "Failed to set number of copies."
End If

This example sets the number of copies to 3 for the "rptCompany" report and checks if the operation was successful.
