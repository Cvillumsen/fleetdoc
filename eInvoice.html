Functions in eInvoice.bas

Function: ChangeStatus(Statusflg As Integer)

The ChangeStatus function updates the status of the selected invoices in the frmImportInvoice form.

Description:

This function checks if the form frmImportInvoice is loaded. If it is, it calls the ChangeSelStatus method from the subform frmImportInvoice_Sub to change the status of the selected records based on the Statusflg parameter.

Parameters:

  • Statusflg: An integer representing the new status to apply to the selected records.

Returns:

This function does not return a value.

Example:

To change the status of selected invoices to 1 (for example, marking them as processed):

Call ChangeStatus(1)
