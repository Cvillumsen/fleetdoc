Function: ins_Casco_Estimate(damageId As Long) As Currency

Returns the estimated cost of a casco (vehicle damage) claim based on the provided damage ID.

Parameters:

  • damageId (Long) - The ID of the damage claim for which the estimate is to be retrieved.

Returns:

Currency - The estimated cost of the casco claim. If the estimate is not found, the function returns 0.


Example:
Dim estimate As Currency
estimate = ins_Casco_Estimate(12345)

Function: ins_Casco_Remaining(damageId As Long) As Currency

Calculates the remaining amount to be booked for casco claims based on the provided damage ID. This is calculated by subtracting the booked and rebilled amounts from the estimated cost.

Parameters:

  • damageId (Long) - The ID of the damage claim for which the remaining amount is to be calculated.

Returns:

Currency - The remaining amount to be booked for the casco claim. If the remaining amount is less than 0, the function returns 0.


Example:
Dim remainingCasco As Currency
remainingCasco = ins_Casco_Remaining(12345)

Function: ins_is_TPL(CoverageType As Integer) As Boolean

Checks if a given coverage type is considered third-party liability (TPL).

Parameters:

  • CoverageType (Integer) - The type of coverage to be checked.

Returns:

Boolean - Returns True if the coverage type is TPL (third-party liability), otherwise returns False.


Example:
Dim isTPL As Boolean
isTPL = ins_is_TPL(4) ' Check if CoverageType 4 is TPL
