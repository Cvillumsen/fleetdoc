Functions in basPaperNameList.bas

Function: GetPaperNameList()

The GetPaperNameList function retrieves and prints a list of paper sizes available for the current printer device.

Description:

This function calls adhGetPaperNames to retrieve a list of paper names (sizes) supported by the current printer. The list of paper sizes is printed to the debug window.

Parameters:

This function has no input parameters.

Returns:

This function does not return a value directly. Instead, it prints the list of paper sizes to the debug window.

Example:

To retrieve and display the available paper sizes for the current printer: 

Call GetPaperNameList()
' Output:
' Letter
' A4
' Legal
' (and other available paper sizes)

Function: GetPaperNames(dr As adh_tagDeviceRec, astrList() As String) As Long

The GetPaperNames function retrieves the list of supported paper sizes (as strings) from the printer driver and returns the number of paper sizes available.

Description:

This function uses the DeviceCapabilities API to query the printer driver for the names of supported paper sizes. The list of names is returned in an array, and the function returns the count of available paper sizes.

Parameters:

  • dr: A structure of type adh_tagDeviceRec that contains the printer device's name and port information.
  • astrList(): An array of String that will be filled with the names of the available paper sizes.

Returns:

Returns a Long representing the number of paper sizes retrieved. If an error occurs, it returns 0.

Example:

To retrieve the paper sizes for a specific printer device: 

Dim dr As adh_tagDeviceRec
Dim paperNames() As String
Dim count As Long

' Assume dr is filled with device details
count = GetPaperNames(dr, paperNames)

If count > 0 Then
    Dim i As Integer
    For i = 0 To count - 1
        Debug.Print paperNames(i)
    Next i
End If
