Functions in basDevNames.bas

Function: adhBuildDevNames(dr As adh_tagDeviceRec) As Variant

The adhBuildDevNames function creates a prtDevNames structure based on the printer's device name, driver name, and port.

Description:

This function is used to build a printer device structure (prtDevNames) by combining the device name, driver name, and port into a format that can be used by the printer system. It also ensures that the device name is within valid length limits.

Parameters:

  • dr: A structure of type adh_tagDeviceRec that contains the device name, driver name, and port information of the printer.

Returns:

Returns a Variant containing the built prtDevNames structure, ready to be assigned to the printer device.

Example:

To build a printer device structure: 

Dim dr As adh_tagDeviceRec
Dim devNames As Variant
' Assume dr is filled with the device information
devNames = adhBuildDevNames(dr)
If Not IsNull(devNames) Then
    Debug.Print "Printer device structure built."
End If

Function: adhGetDevNames(obj As Object) As adh_tagDeviceRec

The adhGetDevNames function retrieves a prtDevNames structure from a form or report and returns it in a adh_tagDeviceRec structure.

Description:

This function extracts the printer device information (e.g., device name, driver name, port) from a form or report's prtDevNames property and stores it in an adh_tagDeviceRec structure.

Parameters:

  • obj: A reference to the form or report from which to retrieve the prtDevNames structure.

Returns:

Returns a filled adh_tagDeviceRec structure containing the printer device information.

Example:

To retrieve the printer device information from a form: 

Dim dr As adh_tagDeviceRec
dr = adhGetDevNames(Forms!frmTest)
If Len(dr.strDeviceName) > 0 Then
    Debug.Print "Device Name: " & dr.strDeviceName
End If

Function: adhParseDevNames(ByVal varDevNames As Variant, dr As adh_tagDeviceRec) As Boolean

The adhParseDevNames function parses a varDevNames string and fills the corresponding fields in an adh_tagDeviceRec structure.

Description:

This function takes a varDevNames string, which contains printer device information, and extracts the device name, driver name, and port. It fills in these details in the adh_tagDeviceRec structure and also returns whether the printer is the default printer.

Parameters:

  • varDevNames: A string containing the printer device information.
  • dr: A structure of type adh_tagDeviceRec to be filled with the parsed information.

Returns:

Returns True if the printer is the default device, otherwise returns False.

Example:

To parse the varDevNames string and extract device details: 

Dim dr As adh_tagDeviceRec
Dim isDefault As Boolean
isDefault = adhParseDevNames(varDevNames, dr)
If isDefault Then
    Debug.Print "This is the default printer."
Else
    Debug.Print "This is not the default printer."
End If
