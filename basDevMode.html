Function: adhRetrieveDevMode(obj As Object, DM As adh_tagDevMode) As Boolean

The adhRetrieveDevMode function retrieves the adh_tagDevMode structure for a given form or report. It is used to extract the printer settings for the specified object.

Description:

This function retrieves the DEVMODE structure, which contains printer settings such as paper size, orientation, and printer-specific features. If the object has no customized printer settings, the default printer's settings are used instead.

Parameters:

  • obj: A reference to a form or report from which to retrieve the DEVMODE structure.
  • DM: A variable of type adh_tagDevMode where the retrieved DEVMODE structure is stored.

Returns:

Returns True if the DEVMODE structure is successfully retrieved, otherwise False.

Example:

To retrieve the printer settings for a form: 

Dim DM As adh_tagDevMode
If adhRetrieveDevMode(Forms!frmTest, DM) Then
    Debug.Print "Device Name: " & DM.strDeviceName
End If

Function: adhSetDevMode(obj As Object, DM As adh_tagDevMode) As Boolean

The adhSetDevMode function sets the prtDevMode property of a form or report based on the provided adh_tagDevMode structure.

Description:

This function allows you to programmatically assign printer settings (e.g., paper size, orientation) to a form or report. It takes the values stored in an adh_tagDevMode structure and applies them to the specified object.

Parameters:

  • obj: A reference to the form or report for which the DEVMODE structure is being set.
  • DM: A variable of type adh_tagDevMode containing the DEVMODE structure to set on the object.

Returns:

Returns True if the DEVMODE structure is successfully applied, otherwise False.

Example:

To set the printer settings for a form: 

Dim DM As adh_tagDevMode
' Assume DM is filled with appropriate values
If adhSetDevMode(Forms!frmTest, DM) Then
    MsgBox "Printer settings applied."
End If

Function: adhGetDefaultDM(dr As adh_tagDeviceRec) As Variant

The adhGetDefaultDM function retrieves the default DEVMODE structure for a given printer device.

Description:

This function retrieves the default printer settings for a device. It is useful when you need to access or modify the default settings (such as paper size, orientation) of a printer.

Parameters:

  • dr: A record of type adh_tagDeviceRec that contains information about the printer device.

Returns:

Returns a Variant containing the default DEVMODE structure for the specified device.

Example:

To retrieve the default DEVMODE structure for a printer: 

Dim dr As adh_tagDeviceRec
Dim DM As Variant
' Assume dr is filled with the device information
DM = adhGetDefaultDM(dr)
If Not IsNull(DM) Then
    Debug.Print "Default DEVMODE structure retrieved."
End If
