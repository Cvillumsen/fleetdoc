Function: GetScrollbarOffset(ByRef frm As Access.Form, sBar As String) As Long

The GetScrollbarOffset function returns the width or height of a form's scrollbars, depending on whether it's a horizontal or vertical scrollbar.

Description:

This function calculates the width (horizontal scrollbar) or height (vertical scrollbar) based on the form's ScrollBars property. It handles different configurations of scrollbars (none, horizontal only, vertical only, or both). If an error occurs, it shows an error message using a message box.

Parameters:

  • frm: A reference to the form from which the scrollbar offset is being retrieved.
  • sBar: A string that indicates whether to retrieve the horizontal ("H") or vertical ("V") scrollbar offset.

Returns:

Returns a Long value representing the scrollbar width for horizontal scrollbars or the scrollbar height for vertical scrollbars, depending on the sBar parameter.

Example:

To get the horizontal scrollbar offset of a form: 

Dim hOffset As Long
hOffset = GetScrollbarOffset(Forms!frmSample, "H")
Debug.Print "Horizontal Scrollbar Offset: " & hOffset

To get the vertical scrollbar offset of a form: 

Dim vOffset As Long
vOffset = GetScrollbarOffset(Forms!frmSample, "V")
Debug.Print "Vertical Scrollbar Offset: " & vOffset

Error Handling:

If an error occurs during execution, the function displays an error message using a message box.
