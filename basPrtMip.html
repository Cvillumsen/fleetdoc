Function: adhRetrieveMIP(obj As Object, MIP As adh_tagMarginInfo) As Boolean

The adhRetrieveMIP function retrieves the margin information (prtMip property) from a specified form or report and fills the provided adh_tagMarginInfo structure.

Description:

This function extracts margin and layout-related settings from the printer properties of a form or report and stores them in a user-defined structure of type adh_tagMarginInfo. The function is useful for accessing and modifying the layout and margin settings programmatically.

Parameters:

  • obj: A reference to the form or report from which to retrieve the margin information.
  • MIP: A variable of type adh_tagMarginInfo that will be filled with the retrieved margin settings.

Returns:

Returns True if the margin information is successfully retrieved, otherwise returns False.

Example:

To retrieve the margin information for a form named "frmTest": 

Dim MIP As adh_tagMarginInfo
Dim fSuccess As Boolean

fSuccess = adhRetrieveMIP(Forms!frmTest, MIP)
If fSuccess Then
    Debug.Print "Items Across: " & MIP.intItemsAcross
Else
    Debug.Print "Failed to retrieve margin information."
End If

Function: adhSetMip(obj As Object, MIP As adh_tagMarginInfo) As Boolean

The adhSetMip function sets the margin information (prtMip property) for a specified form or report using the values stored in a adh_tagMarginInfo structure.

Description:

This function allows you to programmatically set the printer margin and layout settings of a form or report by passing in a adh_tagMarginInfo structure that contains the necessary values. It is useful for automating changes to the layout configuration of printed forms and reports.

Parameters:

  • obj: A reference to the form or report for which the margin information will be set.
  • MIP: A variable of type adh_tagMarginInfo that contains the margin settings to be applied to the form or report.

Returns:

Returns True if the margin information is successfully applied, otherwise returns False.

Example:

To set the margin information for a form named "frmTest": 

Dim MIP As adh_tagMarginInfo
Dim fSuccess As Boolean

' Assume MIP has been filled with values
MIP.intItemsAcross = 2
MIP.lngLeftMargin = 1440 ' 1 inch margin in twips

fSuccess = adhSetMip(Forms!frmTest, MIP)
If fSuccess Then
    Debug.Print "Margin information set successfully."
Else
    Debug.Print "Failed to set margin information."
End If
