Function: adhTrimNull(strString As String) As String

The adhTrimNull function removes any null characters (vbNullChar) from a string returned by the Windows API.

Description:

Sometimes strings returned from Windows API calls may contain embedded null characters (vbNullChar). This function removes the null characters from the string. It is important to ensure that the input string is not Null because the function cannot handle Null values.

Parameters:

  • strString: The string from which to remove null characters.

Returns:

Returns the input string with any null characters removed.

Example:

To trim null characters from a string:

Dim cleanString As String
cleanString = adhTrimNull("Hello" & vbNullChar & "World")
Debug.Print cleanString  ' Output: "Hello"

Function: ANSIToUni(varAnsi As Variant) As Variant

The ANSIToUni function converts an ANSI string to a Unicode string.

Description:

This function uses the StrConv function to convert a string from ANSI encoding to Unicode encoding. It is useful when working with API calls or other processes that require Unicode strings.

Parameters:

  • varAnsi: The ANSI string to be converted to Unicode.

Returns:

Returns the Unicode version of the input ANSI string.

Example:

To convert an ANSI string to Unicode:

Dim unicodeString As Variant
unicodeString = ANSIToUni("ANSI String")
Debug.Print unicodeString  ' Output: Unicode representation of "ANSI String"

Function: UniToAnsi(varUni As Variant) As Variant

The UniToAnsi function converts a Unicode string to an ANSI string.

Description:

This function uses the StrConv function to convert a string from Unicode encoding to ANSI encoding. It is useful when working with legacy systems or APIs that require ANSI strings.

Parameters:

  • varUni: The Unicode string to be converted to ANSI.

Returns:

Returns the ANSI version of the input Unicode string.

Example:

To convert a Unicode string to ANSI:

Dim ansiString As Variant
ansiString = UniToAnsi("Unicode String")
Debug.Print ansiString  ' Output: "Unicode String" in ANSI encoding
